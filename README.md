# A Small-Brief Intro to Space-Time Data in R

Materials for an ATL R Users Group workshop.

In particular, the workshop is to include: 

* Overview of packages (`sp`, `gstat`) and functionality for handling spatial data in R
* How to declare and work with spatial data
* Explanation of the class hierarchy of spatial objects
* Assigning projections (via `rgdal`)
* Performing computations on spatial objects
* Graphing and visualing spatial (and spacio-temporal) data (with `base`, `ggplot2`, `ggmap`, `leaflet`)
